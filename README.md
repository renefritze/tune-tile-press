## Installation and Development

This project uses [uv](https://github.com/astral-sh/uv) for fast, reliable Python package management. To get started:

1. Install uv.

2. Install the package with development dependencies:

```bash
uv sync
```

3. Run tests:

```bash
uv run pytest
```

4. Run linting:

```bash
uv run ruff check .
```
